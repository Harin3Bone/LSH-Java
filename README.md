# Locality Sensitive Hashing (LSH)

## Prerequisites

- Java 25 or higher

## Simulate LSH by IDE

1. Clone the repository to your local machine.
2. Import the project by **Maven** project option in your IDE.
3. Go to `src/test/java/com/cs` packages.
4. Run the `MainTest.java` file to execute the tests for simulate LSH.

## Simulate LSH by command line

1. Clone the repository to your local machine. 
2. Navigate to the project directory in your terminal.
3. Run command `mvn test` to execute the tests for simulate LSH.
